# Icon Generation

The PWA icons need to be generated. You can:

1. Use an online tool like [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
2. Use the provided `icon.svg` in the static folder
3. Generate icons manually:
   - Create `icon-192.png` (192x192)
   - Create `icon-512.png` (512x512)
   - Create `favicon.ico`

Place all generated icons in the `/static` directory.
